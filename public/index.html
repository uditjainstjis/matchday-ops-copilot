    How the decision is made

    1. Generative AI reads the text. Llama 4 Scout on Cloudflare Workers AI extracts a category, a 1–5 severity, a one-line summary and recommended actions, constrained to a JSON schema.
    2. Deterministic policy decides the consequences. Priority, target response time and which units are dispatched are computed by audited pure functions — never by the model. Identical inputs always produce identical dispatch.
    3. It degrades loudly, not silently. If the model is slow, unavailable or returns anything off-schema, an offline keyword classifier answers instead and the result is labelled Degraded — offline classifier.

    Built for PromptWars Challenge 4 — Smart Stadiums & Tournament Operations. Runs entirely on Cloudflare Workers with Workers AI. No incident data is stored.
